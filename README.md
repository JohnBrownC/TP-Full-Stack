# Sitio web de Fotografía

Este proyecto es un sitio web de fotografía creado como parte de un curso de programación. El objetivo principal es mostrar una colección de imágenes y proporcionar funcionalidades básicas para explorar y disfrutar de las fotografías mientras aprendemos a programar.

## Características

- **Galería de imágenes:** Muestra una galería de imágenes donde los usuarios pueden ver y explorar fotografías.

- **Consejos útiles para principiantes:** .


- **Comentarios y opiniones:** Permite a los usuarios enviar comentarios y opiniones sobre las fotografías.

## Tecnologías utilizadas

- HTML5 y CSS3: Para estructurar y diseñar el sitio web.

- JavaScript: Para agregar interactividad y funcionalidad a la página.

- [API de EmailJs]: Utilizada para darle funcionalidad al formulario de contacto.
